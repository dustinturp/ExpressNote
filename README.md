# ExpressNote

## Description
Create and edit notes in the browser. 

![screenshot of App](./public/assets/notetaker_scrn.png)

## Built with 
* node
* ExpressJS
* Jest
* nodemon


## Testing
Clone repo.
`npm i --save-dev jest`
`npm i -D nodemon`
`npm i express`

`npm run dev` to start nodemon live updates.
